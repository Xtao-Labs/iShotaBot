# iShotaBot

## 简介

拥有很多功能的 Python Telegram Bot

## 特色

- 解析 bilibili 视频
- 简易版问与答

## 部署

1. 在 config.gen.ini 中设置 api_id , api_hash
2. 复制 config.gen.ini 为 config.ini
3. 使用 pip 从 requirements 安装所需库
4. ```python3 main.py```
